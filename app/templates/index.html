Travel Model API Documentation

Base URL

The base URL for all API endpoints is: http://0.0.0.0:8000

Endpoints

1. Classification

Classify Location

URL: /classifyLocation

Method: POST

Request Body:

{ "image": file (image file) }

Response:

{ "result": "string" }

Description: Classifies the location in the provided image using a machine learning model.

Possible Results: Bao_Tang_Chung_Tich_Chien_Tranh, Bao_Tang_Lich_Su, Bao_Tang_My_Thuat, Bao_Tang_Thanh_Pho, Ben_Nha_Rong, Bitexco, Bui_Vien_Tay, Buu_Dien_TPHCM, Cau_Mong, Cho_Ben_Thanh, Cho_Binh_Tay, Chua_Ba_Thien_Hau, Chua_Buu_Long, Chua_Ngoc_Hoang, Chua_Phap_Hoa, Chua_Vinh_Nghiem, Cot_Co_Thu_Ngu, Dinh_Doc_Lap, Ho_Con_Rua, Landmark_81, Nha_Hat_Thanh_Pho, Nha_Tho_Duc_Ba, Nha_Tho_Giao_Xu_Tan_Dinh, Thao_Cam_Vien, UBND_TPHCM, Unknown

2. Translation

Translate Text

URL: /translation

Method: POST

Request Body:

{ "text": "string", "option": "string" }

Response:

{ "result": "string" }

Description: Translates the provided text based on the specified option.

Options:

  • "eng-vn": Translate from English to Vietnamese
  • "vn-eng": Translate from Vietnamese to English

Error Handling

All endpoints may return appropriate HTTP status codes for different error scenarios. Common error codes include:

  • 400: Bad Request (e.g., missing required parameters)
  • 500: Internal Server Error

For specific error messages, refer to the response body of the failed request.
